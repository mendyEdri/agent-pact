Read-only tools that let agents explore the protocol state without executing transactions.

## Pact Queries

### get-pact

Get full details of a pact by ID.

| Parameter | Type | Description |
|-----------|------|-------------|
| `pactId` | number | The pact ID |

Returns: buyer, seller, payment, deadline, status, spec hash, verification threshold, stakes, initiator, review period, oracle fee, payment token.

### get-verification

Get verification details for an oracle on a pact.

| Parameter | Type | Description |
|-----------|------|-------------|
| `pactId` | number | The pact ID |
| `oracle` | string | Oracle address |

Returns: score, hasSubmitted, proof hash.

### get-pact-count

Get the total number of pacts ever created.

No parameters.

## Identity

### get-my-address

Get the connected wallet address and ETH balance.

No parameters.

Useful for agents to identify themselves and check available funds before creating or accepting pacts.

### find-oracles

Search for registered oracles by capability (OracleRegistry).

| Parameter | Type | Description |
|-----------|------|-------------|
| `capability` | string | Capability to search for (e.g., `"code-review"`) |

Returns: list of oracle addresses with their stakes and verification counts.

## Reputation

### get-reputation

Look up an address's pact history.

| Parameter | Type | Description |
|-----------|------|-------------|
| `address` | string | The address to check |

Returns: completedAsBuyer, completedAsSeller, disputesLost, totalVolumeWei.

### check-counterparty

Check a counterparty's reputation and get a risk assessment before accepting a pact.

| Parameter | Type | Description |
|-----------|------|-------------|
| `pactId` | number | The pact ID (checks the other party) |

Returns: reputation stats plus a risk level (LOW, MEDIUM, HIGH, NEW).

```
Agent: "Check the counterparty reputation for pact #5
        before I accept it."
```
