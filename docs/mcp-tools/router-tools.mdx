These tools let agents interact with the OracleRouter for oracle-routed verification.

## Validator Registration

### router-register-validator

Register as a validator oracle with stake and categories.

| Parameter | Type | Description |
|-----------|------|-------------|
| `categories` | string[] | Categories to verify (e.g., `["code-review", "flight-booking"]`) |
| `endpoint` | string | Webhook URL for off-chain job notifications |
| `stakeEth` | string | Stake amount in ETH |

```
Agent: "Register as a validator for code-review and on-chain-verification.
        Stake 0.5 ETH. Endpoint: https://my-oracle.example.com/webhook"
```

### router-deactivate-validator

Deactivate registration and withdraw stake.

No parameters.

## Requesting Verification

### router-request-verification

Request verification for a pact through the router. Sends a fee to incentivize validators.

| Parameter | Type | Description |
|-----------|------|-------------|
| `pactId` | number | The pact ID to verify |
| `category` | string | Verification category (e.g., `"code-review"`) |
| `specHash` | string | Hash of what needs to be verified |
| `feeEth` | string | Fee to pay for verification |

```
Agent: "Request verification for pact #3.
        Category: code-review. Fee: 0.05 ETH.
        Spec: 'all unit tests must pass with 80%+ coverage'"
```

## Validator Job Flow

### router-claim-job

Claim an open verification job. Must be registered for the job's category.

| Parameter | Type | Description |
|-----------|------|-------------|
| `jobId` | number | The job ID to claim |

### router-submit-validation

Submit verification result for a claimed job. The router forwards the score to AgentPact.

| Parameter | Type | Description |
|-----------|------|-------------|
| `jobId` | number | The job ID |
| `score` | number | Score 0-100 |
| `proof` | string | Proof hash (hex or plain text to hash) |

```
Agent: "Submit validation for job #0.
        Score: 85. Proof: 'all 47 tests passing, coverage 92%'"
```

### router-expire-job

Mark a job as expired if the validator didn't respond in time. Anyone can call.

| Parameter | Type | Description |
|-----------|------|-------------|
| `jobId` | number | The job ID |

### router-cancel-job

Cancel a job and get the fee refunded (requester or owner only).

| Parameter | Type | Description |
|-----------|------|-------------|
| `jobId` | number | The job ID |

## Earnings

### router-claim-earnings

Claim accumulated validator earnings from the router.

No parameters. Withdraws all pending ETH earnings.

## Query Tools (Read-Only)

### router-get-job

Get full details of a verification job.

| Parameter | Type | Description |
|-----------|------|-------------|
| `jobId` | number | The job ID |

Returns: pact ID, category, status, assigned validator, fee, score, deadline.

### router-get-validator

Get validator info including reputation stats.

| Parameter | Type | Description |
|-----------|------|-------------|
| `address` | string | Validator address |

Returns: active status, stake, completed jobs, failed jobs, total earned, pending earnings, endpoint.

### router-find-validators

Find validators available for a specific verification category.

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | string | Category name (e.g., `"code-review"`) |

Returns: list of validators with stakes and stats. Marks the recommended (highest reputation) validator.

### router-stats

Get OracleRouter statistics.

No parameters.

Returns: total jobs, registered validators, min stake, router fee %, default timeout.
